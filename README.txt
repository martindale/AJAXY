jQuery History Plugin:

Installation:

	Include the follow Javascript:
		<!-- Include jQuery -->
		<script type="text/javascript" src="ajaxy/js/jquery-1.3.2.min.js"></script>
		<!-- Include jQuery History -->
		<script type="text/javascript" src="ajaxy/js/jquery.history.js"></script>

Usage:
	
	To use, refer to:
		ajaxy/demos/history/index.html

jQuery Ajaxy Plugin:

Installation:

	Include the follow Javascript:
		<!-- Include jQuery -->
		<script type="text/javascript" src="ajaxy/js/jquery-1.3.2.min.js"></script>
		<!-- Include jQuery History -->
		<script type="text/javascript" src="ajaxy/js/jquery.history.js"></script>
		<!-- Include jQuery Ajaxy -->
		<script type="text/javascript" src="ajaxy/js/jquery.ajaxy.js"></script>
